You can improove this project on Github

Explaination:

To use the dashboard you must have a Duino-Coin account.

You can create one account HERE

Then just enter your username, no password is requiered. You can now see a loading screen which checks for two connections. a) The Websocket connection to the main server(wss://server.duinocoin.com:15808) and b) a connection to the api file balances.json (https://server.duinocoin.com/balances.json)

With the balances file, the program also checks if your username is valid.

After this "ping check" the main dashboard will load. It gets the data from 3 API files:

  1. https://server.duinocoin.com/balances.json
  2. https://server.duinocoin.com/api.json
  3. https://server.duinocoin.com/miners.json

And reloads them every 3 Seconds to fetch the data.

If you have any other questions, you can contact me in the official Duino-Coin Discord
