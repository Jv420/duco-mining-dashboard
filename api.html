Introduction:

The guide for the official Duino-Coin API's can be read on GitHub.
The dahsboard also provides its own API for more functionality which is listed below. You can use them if:

  • You credit the dashboard visible with it's link (duco.sytes.net)
  • You only use it on websites with less than 1000 clicks per hour.
  • You use them only on websites with non adult content
  • Your website is legally in order and not doing spam, phishing or anything similiar.
In the event of a breach of the rules, the domain (or IP) will be blocked.

API's:

Daily new users

The dashboard gets the number of uses every night and refreshes this JSON file:

https://duco.sytes.net/dailynewusers.json

Server status

Every 6 minutes the duino-coin services are pinged multiple times and a status file gets updated.
node1=pulsepool, node2=starpool, node3=beyondpool, general_master=general tcp ports:

https://duco.sytes.net/ducostats.json

Duco rewards (Beta)

This file is for getting the real duco rewards for each device. It gets its data from the real mined ducos in 15 minutes but is still in beta.

https://duco.sytes.net/ducorewards.json

Background-image

This API creates an background-image based on the parameters submitted. It selects randomly on out of 7 Duino-Coin wallpapers. The scheme is: backgroundimage.php/arg1/arg2 arg1=desktop;mobile arg2=white;dark;normal it is not allowed to crop the image so that the copyright text would no longer be visible.

https://duco.sytes.net/backgroundimage.php/desktop/normal

Faucet informations

For every new donator or faucet user of the dashboard this file gets updated. :

https://duco.sytes.net/faucetinfo.json
